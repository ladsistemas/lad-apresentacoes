Requisitos de Instalação

Guia de especificações para implantação do sistema LAD Food

Servidor LAD Food

Equipamento principal que roda o sistema, banco de dados e serviços.

⚠
Importante: O servidor deve usar obrigatoriamente SSD. Nunca utilize HD convencional — o desempenho do sistema e do banco de dados fica severamente comprometido.
💻

ℹ
O servidor roda Apache Tomcat + Java + PostgreSQL + Impressoras. É a máquina mais exigida do ambiente. As estações (PDVs) são mais leves pois apenas acessam o servidor via navegador.

PDV / Caixa Adicional

Pontos de venda que acessam o servidor via navegador Firefox.

✔
O PDV não precisa rodar o sistema localmente. Ele apenas acessa o servidor pelo navegador, por isso os requisitos são menores.

Pagamento

ℹ
A instalação do totem requer cadastro no serviço PAYGO (para TEF). Serão necessários: Cartão CNPJ, número lógico da maquininha TEF, marca/modelo do PINPAD, e dados do proprietário (CPF, nome, telefone). Abra chamado ao Suporte N2 com essas informações.
